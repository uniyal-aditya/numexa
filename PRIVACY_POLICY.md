# Privacy Policy – Numexa

Numexa collects and stores the following data:
- Server IDs (for settings like counting channels)
- Channel IDs (for counting configuration)
- User IDs (only when required for bot functionality)

What we do NOT collect:
- Messages outside bot commands
- Personal information
- DMs content

Data usage:
- Data is used only to provide bot functionality
- Data is never sold or shared

Data removal:
- Server administrators may request data deletion by removing the bot
